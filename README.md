# Guidr AI Agent
Guidr is a Git-native AI agent designed to help users improve productivity, track progress, and enhance career growth using intelligent workflows.

## Features

### Daily Planner
- Create structured daily plans
- Stores plans using Git-based memory
- Tracks consistency over time

### Progress Tracker
- Analyzes past plans
- Provides feedback on user discipline
- Uses Git as a persistent memory system

### Resume Optimizer
- Improves resume statements
- Enhances clarity and professionalism

## Key Concept
This project demonstrates how Git can be used as a *memory system for AI agents*.
- No external database  
- All data stored in files + commits 
- Tracks user growth over time  

## How It Works
1. User selects an option  
2. Agent processes input  
3. Stores data in /memory  
4. Provides intelligent feedback  
5. Uses Git commits as memory history  

## Tech Stack
- Python  
- Git (for memory)  
- GitAgent structure (agent.yaml, SOUL.md, SKILL.md)

## Use Case
Students and developers can:
- Plan their daily goals  
- Track their consistency  
- Improve resumes  
- Build disciplined habits  

## Git-Native Agent Workflow
- Automatically commits changes after every user action
- Maintains persistent memory using Git history  
- Sunykates GitClaw-style agent execution

## Conclusion
Guidr transforms Git from a version control system into an intelligent memory engine for personal growth and productivy

Note: The agent follows the GitAgent standard structure. Validation is implemented conceptually as per the specification.

This agent simulates GitClaw-style autonomous execution by performing actions, maintaining memory, and committing updates automatically within a Git repository.
