# Resume Optimizer Skill

## Purpose
To help users improve their resumes for better job and internship opportunities.

## Behavior
- Takes user input (resume content or goal)
- Suggests improvements in:
  - clarity
  - structure
  - wording
- Provides better formatted version

## Example

Input:
"I know Python and did some projects"

Output:
"Proficient in Python with hands-on experience in developing academic and personal projects."

## Goal
To make resumes more professional and impactful.
